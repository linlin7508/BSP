**分析結論與科學釋疑**：
- **為什麼 Proposed Full 在 EASY (高密度) 和 HARD (低密度) 密度下 TSR 能完美歸零？**：因為在這些車流分佈中，靜默期與掩護車輛群聚的時機剛好能有效切斷時空連續性。特別是在 **EASY (高密度)** 中，平均捕捉到 **7** 輛車一起參與換假名（隱私熵 2.77），加上 1.0s 的完全靜默，成功讓攻擊者追蹤失敗 (Lost Target)。而在 **HARD (低密度)** 中即便掩護車輛較少 (AS_size=2.00)，由於低密度下攻擊者仍難以在靜默後區分兩台車，依然能達成 0% 的完美防護。
- **中等密度 (MID) 的完美防護**：在 MID 環境中，TSR 同樣達到了完美的 **0.00%**。RSU 啟動時平均召集了 4.27 台車（隱私熵 2.09），結合強制的靜默期與車道清理策略，攻擊者完全失去了追蹤特徵，無法重新鎖定 EV。
- **與 Periodic Sync (定期同步) 的差異**：即使 Periodic Sync 在 MID 中能達到最高的 AS_size (8.11) 與熵值 (2.35)，但缺乏物理層面的靜默期保護，其軌跡幾乎全程暴露 (TSR 在 93%~100% 徘徊)，證明單純換假名而無靜默期的策略在強大攻擊者面前是無效的。

### 啟動成功率定義與符號說明
\[ success\_rate = \frac{N_{success}}{N_{trigger}} \]

### 隱私指標計算公式 (Metrics Definitions)
為了確保實驗的學術嚴謹性，本研究採用的隱私指標定義如下：

1. **TSR (Tracking Success Rate, 追蹤成功率)**:
   \[ TSR = \frac{\text{Correct Links (正確關聯數)}}{\text{Total Relink Attempts (有效追蹤嘗試數)}} \]
   - **有效追蹤嘗試**：指攻擊者在車輛更換假名後，能在 50m 半徑內找到至少一個候選目標的次數。
   - **0% 的意義**：在 Proposed Full 中，由於靜默期的存在，攻擊者在 50m 內找不到任何候選者 (Lost Target)，導致有效嘗試數為 0 或關聯全數失敗，故 TSR 為 0。

2. **AS_size (Anonymity Set Size, 匿名集規模)**:
   - 指在 Mix Zone 啟動瞬間，由 RSU 成功召集並同步更換假名的車輛總數（含 EV 自身）。

3. **Entropy (Privacy Entropy, 隱私熵)**:
   \[ H = \log_2(AS\_size) \]
   - 衡量攻擊者在匿名集中的不確定性。當 $AS\_size = 9.12$ 時，$H \approx 3.19$ bits。

| 符號 | 意義 |
| :--- | :--- |
| **\(N_{success}\)** | 成功召集群聚並啟動混合區的次數 (有啟動紀錄)。 |
| **\(N_{trigger}\)** | EV 經過 RSU 並發起召集評估的總次數 (EV 經過 RSU 總數)。 |
| **\(avg\_AS\_size\)** | 該策略下所有成功啟動之 Session 的平均成員數。 |
| **\(avg\_entropy\)** | 基於 \(AS\_size\) 計算之平均隱私熵 (1/n\(\log_2 AS\))。 |

### 數據判讀分析
- **圖 5-3(a) 隱私熵趨勢 (反直覺現象分析)**：
    - **現象**：實驗數據顯示 \(\theta\) 越高，平均隱私熵與 AS_size 反而略微下降（從 3.19 降至 2.99）。
    - **原因分析**：這反映了 V2X 環境的 **動態交通特性**。當 \(\theta\) 門檻較低（如 2）時，系統能「即時啟動」並捕捉到路口當下最強的群聚波峰；而高門檻（如 5）則要求 RSU 必須「等待」更多車輛進入。在等待過程中，原本處於通訊半徑邊緣的掩護車輛可能已駛離範圍，導致最終啟動時捕捉到的成員數反而較少。
    - **結論**：門檻值 \(\theta = 2\) 不僅啟動容易，更能因「及時性」捕捉到最完整的局部車流，提供最佳隱私掩護。

- **圖 5-3(b) 成功率趨勢**：
    - **現象**：在 Medium Density (50 veh) 環境下，成功率穩定維持在 80%。
    - **分析**：這說明在城市格狀路口，車流群聚是常態性發生的。本研究提出的 Proposed Full 策略展現了極高的系統穩定性，即便在嚴格的門檻下，依然能為 EV 提供充足的隱私保護機會。
